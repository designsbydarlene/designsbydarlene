CHATBOT + GOVERNMENT

OHA CHATBOT
description

01. Context

A chatbot implementation to help alleviate the overwhelming amount of calls received by Office of Hawaiian Affairs (OHA) Front Desk..

02. Solution

Develop a chatbot backed with Google's Dialogflow and Kommunicate's AI dashboard. The chatbot answer frequently asked questions regarding OHA's various support services. And has the ability to connect users with an OHA agent over chat to answer any questions the chatbot is unable to answer.

03. Lessons Learned

I have never created a chatbot prior to the creation of this application. I have learned a lot with regard to setting up keywords, intents, answers, reply tiles, and the flow of dialog when interacting with a chatbot.

04. Apps & Language

Devpost, Visual Studio Code, Google's Dialogflow, Kommunicate.io, Github, Meteor, Bootstrap, Java, HTML, CSS, Photoshop, Galaxy.

05. Contributions

Developed questions, answers, keywords, and intents for the following OHA services: Board Meetings, Emergency Aid, and Grants. Developed the landing page, and created the chatbot avatar.

PETS + EDUCATION

PetPal
description

01. Context

PetPal is a peer to peer adoption service web application.

02. Solution

Develop a mobile application with features that allow for account creation, overview of pets available for adoption, list a pet for adoption, and view and edit their user profile to update listings and/or view their favorited listings.

03. Lessons Learned

Due to the time constraints of the Meteor Hackathon 2021 (5 days), the biggest lesson learned was time-management. Honestly, this project led to many late nights of coding. However, I learned that planning is detrimental to a time-constrained project. Because our group had a clear goal, we were able to implement every feature that we planned for.

04. Apps & Language

Github, Meteor, Semantic UI, Java, HTML, CSS, Photoshop, Galaxy.

05. Contributions

Created prototypes for pages in the application, implemented the tasks in the Github Kanban, created Milestones in Github to break up tasks into a 5-day timeline, designed sign-in and sign-up page, landing page, home page, list a pet page, user and pet profile page, and the learn more page. Attempted some backend and functionality aspects of the application (ended up being completed by other group members).
